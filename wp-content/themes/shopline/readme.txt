Shopline
== Description ==
Shopline is a full featured eCommerce theme. This theme has a clean, modern and selling oriented design which is suitable for creating any type of online store, fashion website and many more. It comes with advance features like typography, category based product slider, product section to display your product in grid layout, testimonial widget section, unlimited color options for each section, three column ad, body background image option, section hide option, custom templates for pages (contact page, blank page, page builder template and no featured image template). Using these templates you can use any page builder plugin available for WordPress (Like: SiteOrigin, Beaver Builder, Elementor and many more). Theme contain seven widget area (Four for footer, Two for sidebar (default pages) and one for shop page). Theme is specially made for WooCommerce plugin. And lead form builder is integrated. Through which you can easily collect your leads. Theme is compatible with Wishlist, bbPress, Contactform7 and with many other WP plugin.
