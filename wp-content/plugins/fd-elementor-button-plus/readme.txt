Additional Styles and Options for elementor Button Widget.


== Description ==

Additional Styles and Options for elementor button Widget. This plugin adds new elements/widgets to Elementor Page Builder.

<a href="https://elementordemo.flickdevs.com/elementor-button-plus/" target="_blank">Demo</a>

Note: This plugin is an add-on of Elementor Page Builder(https://wordpress.org/plugins/elementor/) and will only work with Elementor Page Builder installed.

Feel free to <a href="https://flickdevs.com/contact-us/" target="_blank">contact us</a> if you find there are no elementor add-on/widget available that match your requirements. We are available to create it for you.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
